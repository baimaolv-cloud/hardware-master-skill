# 硬件评分标准参考手册

> 本文档为 hardware-master-skill 使用的评分基准，基于 **2026 年主流硬件市场行情**。
> 每个维度满分 100 分，综合评分取六项加权平均（或简单平均）。

---

## 权重说明（默认均等）

| 维度 | 权重 | 说明 |
|------|------|------|
| CPU | 25% | 计算核心，影响全局性能 |
| 内存 | 20% | 多任务、编译、AI 推理关键 |
| 存储 | 15% | 读写速度与容量 |
| 显卡 | 25% | 图形/AI 加速，2026 年权重提升 |
| 屏幕 | 10% | 色准、分辨率、亮度 |
| 保值度 | 5% | 二手市场行情 |

---

## 1. CPU 评分基准

| 分数区间 | 参考机型（2026 年行情） | 说明 |
|---------|----------------------|------|
| 95–100 | Apple M4 Ultra / AMD Threadripper 9000 | 顶级工作站级别 |
| 85–94  | Apple M4 Pro (12C) / Intel Core Ultra 9 285H / AMD Ryzen AI Max 395 | 旗舰移动/桌面 |
| 75–84  | Apple M3 Pro / Intel Core i9-14900HX / AMD Ryzen 9 7945HX | 高性能移动端 |
| 65–74  | Apple M2 / Intel Core i9-12900H / AMD Ryzen 9 6900HX | 上一代旗舰 |
| 55–64  | Intel Core i9-9980HK / AMD Ryzen 7 5800H | 2019–2020 旗舰，2026 年中端偏上 |
| 40–54  | Intel Core i7-10th Gen / AMD Ryzen 5 5600H | 主流上代 |
| <40    | 更早期 / 低端处理器 | 性能明显不足 |

**评分参考指标（可联网查询）：**
- Cinebench R23 多核 / 单核
- Geekbench 6 多核 / 单核
- CPU Benchmark（PassMark）排名

## 2. 内存评分基准

| 分数区间 | 容量 | 说明 |
|---------|------|------|
| 90–100 | 64GB+ (LPDDR5 / DDR5) | 专业工作站级别 |
| 80–89  | 32GB (LPDDR5 / DDR5 / 统一内存) | 2026 年主流高端 |
| 70–79  | 32GB (DDR4) | 容量足但带宽受限 |
| 55–69  | 16GB (DDR5/LPDDR5) | 普通用户够用 |
| 40–54  | 16GB (DDR4) | 偏低，重度多任务吃力 |
| <40    | 8GB 及以下 | 2026 年已不推荐 |

## 3. 存储评分基准

| 分数区间 | 规格 | 说明 |
|---------|------|------|
| 90–100 | 2TB+ NVMe PCIe 4.0/5.0，读 > 5GB/s | 顶级速度 |
| 80–89  | 1TB NVMe PCIe 4.0，读 3–5GB/s | 主流高端 |
| 70–79  | 1TB NVMe PCIe 3.0，读 2–3GB/s（如 Apple AP1024N） | 较好 |
| 55–69  | 512GB NVMe / 1TB SATA SSD | 速度或容量偏低 |
| <55    | HDD / eMMC / SATA SSD 512GB 以下 | 明显落后 |

## 联网评分补充说明

执行评分时，建议：
1. 通过 web_search 查询目标 CPU/GPU 的最新 benchmark 排名
2. 查询同型号二手市场价格（闲鱼、eBay 等）
3. 对照上述区间给出分数，并说明依据
4. 若硬件信息不完整，优先运行 `scripts/collect_hardware.sh` 补采
